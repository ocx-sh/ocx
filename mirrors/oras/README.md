# ORAS

ORAS (OCI Registry As Storage) is a command-line tool for managing content stored in OCI-compliant registries. It enables pushing, pulling, copying, and discovering artifacts — not just container images — using any OCI registry as a general-purpose storage backend.

## What's included

- **oras** — push, pull, copy, attach, discover, and manage OCI artifacts
