# Lychee

Lychee is a fast, async, stream-based link checker written in Rust. It finds broken URLs and mail addresses inside Markdown, HTML, reStructuredText, websites and more. It handles HTTP(S), mailto, relative file paths, and anchor fragments, and ships a GitHub Action for CI integration.

## What's included

- **lychee** — the link checker CLI
- **lychee.1** — man page (under `docs/` in the archive)
- Supplementary docs: README, TROUBLESHOOTING, PRE_COMMIT guide
