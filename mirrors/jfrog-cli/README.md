# JFrog CLI

JFrog CLI (`jf`) is a compact client that provides a simple interface for automating access to the JFrog Platform. It supports managing artifacts in Artifactory, scanning with Xray, and integrating with CI/CD pipelines. Built in Go, it ships as a single self-contained binary.

## What's included

- `jf` — the JFrog CLI executable (manages Artifactory, Xray, Distribution, Pipelines, and more)
