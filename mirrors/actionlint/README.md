# actionlint

actionlint is a static checker for GitHub Actions workflow files. It catches syntax errors, type mismatches in expressions, misconfigured runners, missing permissions, and dozens of other common workflow mistakes — both at the YAML level and in shell snippets embedded in `run:` blocks (via shellcheck).

## What's included

- **actionlint** — workflow linter CLI
