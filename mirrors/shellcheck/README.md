# ShellCheck

ShellCheck is a static analysis tool that finds bugs in Bash, sh, and other shell scripts. It points out syntax issues, semantic problems, and subtle pitfalls, and suggests corrections. ShellCheck can be used from the command line, integrated into editors, or run in CI pipelines.

## What's included

- **shellcheck** — static analysis and linting for shell scripts
