# Bats

Bats (Bash Automated Testing System) is a TAP-compliant testing framework for Bash and POSIX shell. Tests are written as ordinary shell scripts using a lightweight `@test "..."` directive. bats-core is the actively maintained fork that has succeeded the original repo.

## What's included

- **bats** — the test runner (`bin/bats`)
- **bats-format-tap**, **bats-format-junit**, **bats-format-pretty** — output formatters
- **libexec/** — supporting helpers (`bats-exec-test`, `bats-preprocess`, etc.)
