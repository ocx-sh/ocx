# uv

uv is an extremely fast Python package and project manager, written in Rust by [Astral](https://astral.sh). It serves as a drop-in replacement for pip, pip-tools, pipx, poetry, pyenv, twine, virtualenv, and more — unified into a single tool with 10-100x better performance.

## What's included

This package provides two command-line tools:

- **uv** — Python package installer, resolver, project manager, and Python version manager
- **uvx** — Run tools published to PyPI without installing them (like pipx)
