# Node.js

Node.js is a cross-platform JavaScript runtime built on Chrome's V8 engine. It enables server-side JavaScript execution and comes with npm, the largest ecosystem of open-source libraries.

## What's included

- **node** — JavaScript runtime
- **npm** — package manager
- **npx** — package runner
- **corepack** — manager for package managers (Yarn, pnpm)
